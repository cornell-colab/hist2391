This repository includes all of the data and files for IIIF manifests and images processed by the Digital CoLab for HIST 2391/AMST 2391: From Terra Incognita to Territories of Nation-States: Early American History in Two Dozen Maps. All files were processed by Kiran Mohammadi-Williams, so if you have questions about the contents, please reach out to her.
Images were processed using Glen Robson'sIIIF Tiler and IIIF Workbench, GitHub, and Bodleian IIIF Manifest Editor.
Please don't change any of the names of directories and files in this repo or the name of the repo itself because it will break everything.
